## Case 1 – Windows Media Creation Tool Error

### Environment
- Windows laptop
- Objective: Download Windows ISO for VirtualBox lab

### Problem
Media Creation Tool failed to start during setup.

### Troubleshooting Steps
1. Restarted system
2. Re-downloaded tool
3. Ran as administrator

### Resolution
Used alternative method to obtain Windows ISO.

### Learning Outcome
Installation tools can fail due to permissions, system configuration, or environment. Always test alternative solutions.
